        <p>
This project explored how social anxiety influences cognitive performance and physiological responses under stress. Using a combination of behavioral experiments and wearable sensing, we investigated how individuals with varying levels of social anxiety perform on working memory tasks involving emotional facial expressions. Participants completed cognitive tasks under both stressful and non-stressful conditions while their physiological responses were monitored using electrodermal activity (EDA) sensors.
